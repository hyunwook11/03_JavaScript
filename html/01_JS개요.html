JavaScript

Script 언어

        기본 프로그램의 동작을
        사용자의 요구애 맞게 수행되되도록 해주는 언어

        별도의 소스코드 컴파일(한 번에 이진법 변환)을 하지 않고
        
        인터프리터를 이용해 한 줄씩 번역, 수행하는 언어

        - 장점 : 컴파일에 소요되는 시간이 없음
        -단점 : 프로그램 실행 시간이 상대적으로 느림

JavaScript란?

        웹 브라우저(크롬 등...)에서
        주로 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어

        ECMA(유럽 컴퓨터 제조업 협회)
        Script 표준을 따르는 대표적인 웹 기술
        (JS 버전명 : ES5, ES6, ...)

JavaScript를 작성하는 방법

  1. 브라우저 개발자도구 -> Console에 입력하기
    (JS 연습, 확인 등에 많이 사용하길 권장)
  2. HTML 문서 내부에 작성하는 방법(Script 태그 이용) -> internal 방식
  3. HTNL 문서 외부에 존재하는 .js 파일을 연결해서 사용 -> External 방식 
                    1) 원하는 위치에 .js 파일 생성
                2) 연결을 원하는 HTML 파일 body태그 제일 아랫쪽에
                script태그를 작성 ->src 속성 추가
  4. 태그에 직접 JS 코드를 작성 -> inline 방식(권장 X)
