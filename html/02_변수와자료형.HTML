변수 (variable)

        - 변수란? 메모리 값을 저장할 수 있는 공간
        (RM, 메모리 : 주기억장치)

        - 메모리에 기록된 값, 코드만 프로그램 실행에 사용 가능
        - 변수에 원하는 이름을 붙이고 값을 대입(=) 
        - 필요한 곳에서 "변수명"을 호출하면 해당 변수에 저장
        된 값을 가져와 사용 가능

        - 변수에 저장된 값을 언제든지 변경 가능
        (단, 상수는 제외)

변수명 명명 규칙

        -의미 있는 이름을 작성(대충 a,b 이런 형태로 지양)
        - 대, 소문자 구분 가능
        -특수 문자는 _만 사용가능
        - 숫자도 사용 가능 ("단 첫 글자는 제외")
        카멜케아스로 작성]
        -> 여러 단어를 연달아 작성
