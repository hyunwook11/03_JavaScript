Early Return

        return; : 현재 실행 중인 함수를 "즉시 종료" 하고
                  호출한 곳으로 돌아감
        
        if문 조건에
        현재 수행하려는 기능(함수)의 예외 상황을
        조건식으로 작성하여
        예외 상황 발생 시 즉시 함수를 종료(return;)하게 만드는 방법

        장점 : 코드 들여쓰기가 줄어들고 간결해짐
               유지보수성 향상
        단점 : return 구문에 함수 내에 분산되어
               함수 코드의 복잡도가 늘어난다

두 수 A,B를 입력 받아 A가 B의 배수가 맞는지 확인

        [조건]
        1) A, B 중 하나라도 입력되지 않을 경우
        -> "입력되지 않은 값이 존재합니다"

        2) A가 B 보다 작을 경우
        -> A가 B보다 작습니다

        3) A가 B보다 클 경우
        -> 배수 판별 수행
A :
B :

switch, case, default

        switch문
        - 식 하나의 결과로 많은 경우의 수를 처리할 때
        사용하는 조건문

        - 식의 결과와 일치하는 값을 가진 case 구문이 수행됨
        == if/else if(A === B) 형태와 유사

        - 식의 결과와 일치하는 case가 없을 경우 default 구문 수행
        == else와 유사
        
        
            switch(식){ // 식의 결과 값은 정수 또는 문자열

                case 결과1 : 결과 1의 수행 코드; break;
                case 결과2 : 결과 2의 수행 코드; break;
                case 결과3 : 결과 3의 수행 코드; break;

                ...
                default : 모든 case가 만족하지 못하는 경우 수행

            }

입력된 수가 3,4,5가 맞는지 확인
